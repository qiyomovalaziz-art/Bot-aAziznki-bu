# Azizbek Curipto Bot

Flask + Telegram bot for Railway deployment.

## Deployment steps
1. Create a new GitHub repo and upload these files.
2. Connect repo to [Railway](https://railway.app).
3. Go to `Variables` → Add new variable:
   - Key: `BOT_TOKEN`
   - Value: (your Telegram bot token)
4. Deploy project.
